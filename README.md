## Introduction

Welcome to my personal portfolio website!

This portfolio is a collection of the work I've built, the technologies I've explored, and the journey I'm on as a Software Engineer and aspiring Data Science / AI graduate student. It brings together my projects, technical writing, achievements, certifications, and ways to connect with me—all in one place.


## Features

* **Projects** – Explore software engineering, AI, cloud, and machine learning projects.
* **Technical Blogs** – Read articles where I simplify concepts, share tutorials, and document my learning journey.
* **Experience** – Learn about my professional experience, internships, and technical skills.
* **Certifications** – View certifications across cloud computing, AI, and software development.
* **Resume** – Access my latest resume directly from the portfolio.
* **Contact** – Connect with me through LinkedIn, GitHub, or email.
